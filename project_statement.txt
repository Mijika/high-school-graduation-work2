프로젝트 명세서

작업 종류: 2학기 졸업과제
프로젝트명: 부엉이는 바위에서 "우흥~"하고 웁니다.
팀명: NMH - 너무(노무) 멋진 핸썸 가이)

요구 사항
파이썬을 이용하여 라즈베리파이를 제어하여 바위 위에 있는 부엉이가 움직이면서 지정한 소리들을
출력해야한다.

기능
1. 지정된 소리들이 나와야한다.
2. 부엉이가 90' 방경으로 계속해서 움직일수 있어야 한다.

3. 시스템 시작후 5초 후 '우흥'등 음원 파일을 실행후 파일에 맞추어 부엉이가 원호 운동을 한다.
4. 각 사용 시나리오에 맞게 음성파일과 모션을 변경한다.

하드웨어 명세
1. 라즈베리파이를 사용한다.
2. 서보 모터 2개를 이용하여 부엉이를 조작한다.
2.2 서보 모터는 가능하면 외부 전원을 사용할 수 있게 구성한다.
3. 바위의 구성 요소는 내부는 스트로폼을 외부는 찱흙과 기타 조형물을 이용한다.
4. 220V 전기 콘서트를 이용하여 전원으로 사용한다.
5. 스티커는 AUX라인과 USB전원을 사용할 수 있는 스티커를 사용한다.
6. 부엉이는 인형을 사용하여 무게를 줄인다.

소프트웨어 명세
1. 파이썬을 사용한다.
2. 멀티 쓰레드를 이용하여 음원파일과 모션을 제어한다.
3. 객체지향적으로 설계해야 한다.
